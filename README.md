# pos-system
Real-Time POS &amp; Inventory Management System built with Laravel 12, PHP 8.3, Pusher WebSockets, Tailwind CSS, Alpine.js, Spatie Permission, DomPDF &amp; Maatwebsite Excel. Features barcode scanning, role-based access (Admin, Manager, Cashier), live stock sync, PDF invoices, and multi-payment support (Cash, Card, bKash).
